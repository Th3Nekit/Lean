## Что это

Lean — лёгкий VPN-клиент для Android с упором на скорость и обход блокировок.
Никакой рекламы, аккаунтов и слежки — импортируешь подписку или конфиг и подключаешься.

## Возможности

- **Протоколы:** VLESS/Reality, VMess, Trojan, Shadowsocks, Hysteria2, Hysteria, TUIC, WireGuard, **AmneziaWG** (обход DPI).
- **Подписки** и share-ссылки, импорт `.conf`, deep-link `lean://`.
- **Material You** — тёмная / светлая / AMOLED темы, выбор цвета акцента.
- **Плитка** быстрых настроек, авто-выбор быстрейшего сервера, пинг серверов.
- **Раздельный туннель** по приложениям, kill-switch, выбор DNS, обход локальной сети.
- Мультиарх: `arm64-v8a`, `armeabi-v7a`, `x86_64` + универсальный APK.

## Установка

1. Открой вкладку [**Releases**](../../releases/latest).
2. Скачай APK: бери `universal`, если не уверен в архитектуре, либо нужный `arm64-v8a` / `armeabi-v7a`.
3. Разреши установку из этого источника и установи.
4. Импортируй подписку/конфиг и подключайся.

> ⚠️ **iOS пока не поддерживается** — клиент только для Android.
